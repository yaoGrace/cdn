phpGrace兼顾运行效率与开发效率的开源PHP框架

phpGrace 是一个免费开源的，快速、简单的轻量级PHP开发框架，兼顾运行效率与开发效率的开源php框架。phpGrace 遵循 Apache2 开源许可协议发布，您可以免费使用 phpgrace，更允许您把基于 phpGrace 框架开发的应用开源或商业产品发布/销售。
框架特性
  • 轻快
    phpGrace 体积小，运行效率高！初始化速度可达1毫秒！优秀的框架不应该负重前行、费力启动，phpGrace 给您无比的轻快！
  • 更精致的 MVC
    phpGrace 在遵循 MVC 模式的基础上，大幅发简化了系统的复杂度，更合理的项目规划、更容易维护及拓展的目录结构！
  • 十年铸剑
    phpGrace 是十年 php 及 php框架经验的精华！在此框架发布前已经经历了6个重要版本的更新，同时也经过上百个项目的严格检验！
  • 更全的工具库
    数据库、MD5、验证码、CURL、上传、下载、分页、二维码... phpGrace 拥有全面且强大的工具库，开发不在苦恼而是快乐！
  • 天然模板
    php 天然就是一款模板引擎，grace 采用原生 php 语法作为视图引擎，您不必再去学习模板语法！系统运行效率极大提高！
  • 优秀的缓存机制
    phpGrace 封装了优秀的缓存组件，支持: file、memcache、redies，配置简单、使用便捷、切换便捷，为高效开发奠定坚实基础！
