SMTP 邮件发送


功能描述
mailer 类基于 phpmailer，通过简单的配置即可完成基于 smtp 方式的邮件发送。

php 扩展需求

 开启 php_openssl 扩展


部署说明
下载 phpMailerTool.zip 解压后将mailer.php文件及phpmailer文件夹直接部署到 phpGrace/tools 文件夹下。

配置说明
在您需要调用的控制器 里面，正确填写以下配置：

调用phpmailer,创建对象 
$mailer = new \phpGrace\tools\mailer();
是否开启调试模式 
$mailer->debugIsOpen(false) ; //参数： true 表示开启调试 ，false表示关闭调试。 默认关闭debug调试模式
例1：非ssl加密方式配置 
 // 例如配置阿里云邮箱参数
$ConfigData = array(
	'Host'       => 'smtp.aliyun.com', //smtp 服务器地址
	'Port'       => 25,           //smtp 端口
	'FromName'   => 'xiaoyaoGrace',    //自定义发件人昵称
	'Username'   => 'xxxxx@aliyun.com', //邮件账户
	'Password'   => 'xxxxxx'       //对应密码
);  
// 设置参数
$mailer->setMailConfig($ConfigData);
例2：ssl加密方式配置 
// 例如配置QQ邮箱参数
$ConfigData = array(
	'Host'       => 'smtp.qq.com', //smtp 服务器地址
	'Port'       => 465,           //smtp 端口
	'SMTPSecure' => 'ssl',		   //加密方式SSL	
	'FromName'   => 'xiaoyaoGrace', //自定义发件人昵称
	'Username'   => 'xxxx@qq.com', //邮件账户
	'Password'   => 'xxxxxxxxxxxxxx' //QQ邮箱不是用密码，而是用授权码
);
// 设置参数
$mailer->setMailConfig($ConfigData);


发送邮件代码(控制器内)

//  实例化邮件发送对象
$mailer = new \phpGrace\tools\mailer(); 
// 配置邮箱
$ConfigData = array(
	'Host'       => 'smtp.qq.com', //smtp 服务器地址
	'Port'       => 465,           //smtp 端口
	'SMTPSecure' => 'ssl',	 // 非 ssl 加密 可以不写这项 ，QQ邮箱需要填写
	'FromName'   => 'xiaoyaoGrace',    //自定义发件人昵称
	'Username'   => 'xxxx@qq.com', //邮件账户
	'Password'   => 'xxxxxx'       //对应密码 ,如QQ邮箱 ，需要填写授权码 
);
// 是否开启debug
$mailer->debugIsOpen(true) ;
// 设置参数
$mailer->setMailConfig($ConfigData);

// 收件人地址，多个添加数组元素即可
$mailAddress = array("liukuaizhuan@qq.com");
//  邮件标题
$subject   = "来自 phpGrace 的通知";
//  邮件内容
$mailBody    = " 来自 phpGrace 的通知!，欢迎使用 phpGrace ";
//附件【可以不设置，使用数组方式设置附件】
$addAttachments = array('2.php', '1.jpg');
//使用send方法发送邮件
//参数：1 收件人地址 2 标题 3 内容 4 附件【可选参数 默认 null】
$res =  $mailer->send($mailAddress,$subject,$mailBody,$addAttachments);
// 返回结果 ，成功返回 1，失败返回空
echo $res;
