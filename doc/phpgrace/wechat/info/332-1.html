消息数据日志

开启数据日志记录

打开文件 graceWeChat.php ，将 GWECHAT_LOG 常量的值改成 true 即可开启数据日志记录功能。


查看日志

在浏览器中访问日志文件即可查看数据，日志位置：

http://您的服务器地址/log.html

日志格式

时间: 2017-06-16 10:08:27
微信原始数据：{"ToUserName":"gh_b663823c3664","FromUserName":"op15jsy7D14d8BAvDTTLTdumuz_I","CreateTime":"1497578907","MsgType":"text","Content":"\u89e3\u51b3","MsgId":"6432052429157543457"}
解析后的数据：
ToUserName : gh_b663823c3664
FromUserName : op15jsy7D14d8BAvDTTLTdumuz_I
CreateTime : 1497578907
MsgType : text
Content : 解决
MsgId : 6432052429157543457


生产环境关闭数据记录避免IO开销

define('GWECHAT_LOG',        'false');
