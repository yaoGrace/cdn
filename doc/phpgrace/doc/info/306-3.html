视图创建

视图概述
视图作为MVC模式中的终端环节，在web编程中负责dom结构的展示。grace 使用 php作为“天然模板“，您不必再去学习枯燥无聊的模板语法，系统更不必浪费资源去完成复杂的解析工作！

视图文件创建及命名规则

视图文件命名 :
1、控制器名称_方法名称.php //此种命名系统会进行自动展示
2、视图名称.php //需要手动调用
视图文件位置 : /分组文件夹/views/对应视图文件


控制器与视图间的变量传递
您定义的视图在控制内被自动或者使用 $this->display() 调用，其实调用的核心非常简单且高效，她就是“include”，这样就意味着您的视图文件和控制器都在一个对象内部！那么......
他们可以共享成员变量，即 $this->变量名称！

演示代码：

//控制器代码 :
<?php
class indexController extends grace{
    public function index(){
        $this->webInfo = array(
           'phpGrace',
            'phpGrace, PHP框架',
            'phpGrace 轻快的实力派！'
        );
        $this->name = 'grace';
        //变量可以来自数据库查询哦 ^_^
        $this->data = array(
            array('name' => '张三', age => 18),
            array('name' => '李四', age => 20),
        );
    }
}
//视图文件 index_index.php 代码
<html>
<head>
<title><?php echo $this->webInfo[0];?></title>
</head>
<body>
<h1><?php echo $this->name;?></h1>
<ul>
    <?php foreach($this->data as $rows){?>
    <li><?php echo $rows['name'];?> : <?php echo $rows['age'];?></li>
    <?php }?>
</ul>
</body>
</html>
